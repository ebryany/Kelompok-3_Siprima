## 🚀 **STEP-BY-STEP IMPLEMENTATION**

### **Step 1: Create Folder Structure**
```bash
# Buat folders di src/icon/
mkdir src/icon/auth
mkdir src/icon/navigation  
mkdir src/icon/actions
mkdir src/icon/status
mkdir src/icon/priority
mkdir src/icon/users
mkdir src/icon/reports
mkdir src/icon/forms
mkdir src/icon/system
```

### **Step 2: Download Icons**
```bash
# Download dari Tabler Icons, Heroicons, atau Feather
# Rename sesuai dengan naming convention di atas
# Place di folder yang sesuai
```

### **Step 3: Update IconLoader**
```bash
# Update Utils/IconLoader.java dengan mapping lengkap
# Compile dan test
javac -cp . Utils/IconLoader.java
java -cp . Utils.IconLoader
```

### **Step 4: Implementation ke Forms**
```java
// Contoh implementation di FormLogin:
import Utils.IconLoader;

// Di constructor atau setupIcons():
btnLogin.setIcon(IconLoader.Icons.LOGIN);
btnBack.setIcon(IconLoader.Icons.BACK);
txtUsername.setLeadingIcon(IconLoader.Icons.USER);
txtPassword.setLeadingIcon(IconLoader.Icons.LOCK);
```
